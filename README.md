# 向东渠事迹数字馆新版原型

阶段 1 原型：Next.js + TypeScript + Tailwind CSS 前台页面骨架。

## 当前范围

- 首页
- 数字导览
- 历史资料
- 精神内涵
- 时代价值
- 资讯动态
- 视频展播
- 后台入口

阶段 1 只使用本地 mock 数据和静态页面结构，不包含数据库、登录、上传、真实后台 CRUD 和部署。

## 技术栈

- Next.js App Router
- TypeScript
- Tailwind CSS
- 本地 mock 数据

## 本地运行

```bash
npm run dev
```

默认访问地址：

```text
http://localhost:3000
```

## 后续方向

后续阶段可以继续完善前台展示、整理统一内容结构、增加后台管理界面，再接入 API、数据库和素材上传能力。
